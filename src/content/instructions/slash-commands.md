Les commandes slash sont des raccourcis qui déclenchent des actions spécifiques dans Copilot.

## Commandes principales

- **/clear** - Démarre une nouvelle session de chat
- **/explain** - Explique le code sélectionné
- **/fix** - Corrige les bugs dans le code
- **/fixTestFailure** - Trouve et corrige un test défaillant
- **/tests** - Génère des tests pour le code
